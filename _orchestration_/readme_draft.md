# Team11 Project

## 🥇 Idea: Life Soundtrack

**Angle:** Music as a new communication medium, not music as entertainment.

**Tagline:** "Turn any life moment into a song you'll keep forever."

### Demo Flow

1. **User uploads:** Photos, journal entry, voice notes?
2. **AI identifies:**
   - Key memories
   - Emotional arc
   - Important people
3. **Claude generates:**
   - Lyrics
   - Music style matching the emotion
4. **App publishes:** User's input plus a song that fits the input

### Examples

| Life Moment | Generated Song Style |
|---|---|
| Graduation | Uplifting anthem |
| Breakup | Indie acoustic song |
| Wedding | Cinematic soundtrack |
